# ABCDeal

ABCDeal is a student project – a website and admin panel for car dealers. It’s mostly a code demo (Node/Express backend, MongoDB, React frontend). 

---

## Quick Overview
- **Features:**
  - JWT login/registration
  - Complaint and contact forms
  - Admin panel to view and manage data
- **Stack:** Node.js, Express, MongoDB Atlas, Cloudinary, React, Tailwind CSS

## Technology Choice: Why Node.js?
We chose Node.js with Express for the backend of this project because it provides a fast, flexible, and modern development workflow that fits our needs well.
